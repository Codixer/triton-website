#### Why Triton?

Do you own an international Minecraft server? Would you like to send each player, messages in their own language?
Well, look no further! Triton offers this among a whole host of other awesome features! This plugin uses a placeholder which will be replaced (dynamically, per player) with a message defined in the language file.

Triton works with any plugin that allows customization of messages.

#### Features

**IMPORTANT NOTE:** Triton is **not** an automatic/machine translator. **All** messages must be configured manually in the desired language.

<FeatureItem>
  Translate any chat message, action bar, title/subtitle, inventory titles, scoreboards, holograms,
  NPC names, kick/ban messages, tab header/footers, custom tabs, item names/lores, signs and
  bossbars<sup>2</sup>.
</FeatureItem>
<FeatureItem>Unlimited languages</FeatureItem>
<FeatureItem>Dynamic values in translated messages</FeatureItem>
<FeatureItem>BungeeCord support</FeatureItem>
<FeatureItem>API for developers</FeatureItem>
<FeatureItem>Language GUI selector with flags</FeatureItem>
<FeatureItem>Edit your messages online with TWIN (<strong>T</strong>riton <strong>W</strong>eb <strong>In</strong>terface)</FeatureItem>
<FeatureItem>Reload your config in-game without restarting your server</FeatureItem>
<FeatureItem>Change how the placeholder looks depending on the type of message (chat, scoreboard, item, etc)</FeatureItem>
<FeatureItem>Change another player's language using commands</FeatureItem>
<FeatureItem>Run commands when a player changes their language</FeatureItem>

<Typography variant='caption'>
  <sup>2</sup> Bossbar translation requires Spigot 1.9+
</Typography>
